Display the current Bitstamp Bitcoin Price in Realtime. The Price updates automatically.

== Description ==

Just insert the shortcode `[bitstamp_price]` wherever you want to display the current Bitcoin price in realtime.

Data Source: [Bitstamp Websocket](https://www.bitstamp.net/websocket/)

**Features of this Plugin**

* Realtime Price
* Compatible with every Theme
* Easy to use
* Customizable Colors


= The Shortcode =

In order to embed the Bitcoin Price on a new post/page and use the following shortcode:

`[bitstamp_price]`

= Customizations (Optional) =

Your can force custom size and color of the price.

Example: `[bitstamp_price size="24px" color="#d3d3d3"]`


== Installation ==

1. Upload `bitstamp-bitcoin-price` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin from Admin > Plugins menu.
3. Once activated your site is ready!

== Frequently Asked Questions ==

= Which Symbols are supported =

Only BTC/USD at the moment.

= Does this player work on iPhone? =

Yes. It works on all mobile devices.

= Can I use it in Widgets? =

Yes. Also works in widgets.
